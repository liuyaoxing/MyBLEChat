# 左右手
## 左右手可以通过无线热点和蓝牙聊天传送文件，操作简单。
#### 左右手是一款局域网内即时通信软件, 基于蓝牙传输协议。
#### 不需要服务器支持。
#### 支持文件/文件夹的传送 (支持大文件传送[4G以上]), 发送方和接收方都可查看传送进度功能。
#### 共享文件功能。
#### 用户上下线进行通知功能。
#### 蓝牙传输、蓝牙聊天。
#### 支持无线热点文件传输、聊天，快到不行。
